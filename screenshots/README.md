# Screenshots

Tambahkan screenshot aplikasi di folder ini dengan nama:

- `kiosk-menu.png` - Tampilan menu kiosk
- `kiosk-cart.png` - Keranjang belanja
- `kiosk-payment.png` - Pilihan pembayaran
- `dashboard.png` - Admin dashboard
- `menu-management.png` - Manajemen menu
- `reports.png` - Laporan penjualan
- `kitchen.png` - Kitchen display
- `kasir.png` - Kasir view
- `settings.png` - Settings page

**Recommended size:** 1920x1080 atau 1280x720
